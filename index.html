Ultrasonic distance sensor

An ultrasonic distance sensor is an electronic device used to measure the distance between itself and an object using sound waves. It works on the principle of echolocation, where the sensor emits high-frequency ultrasonic waves (beyond human hearing) and then listens for the echo reflected back from an object. By calculating the time it takes for the echo to return, the sensor determines the distance.
